# Retail & Warehouse Sales Dashboard (2020)

A comprehensive sales analysis and interactive dashboard for a retail and warehouse beverage distribution company.

## 📊 Project Overview

This project analyzes **30,000+ sales records** from 2020 for a beverage distributor dealing in Beer, Wine, Liquor, and related products. The analysis focuses on retail vs warehouse sales performance, category-wise trends, top performers, and seasonal patterns.

### Key Insights
- **Beer** dominates sales (73.4% of total revenue)
- Warehouse sales contribute **79.81%** of total business
- Strong seasonal peak in **July** (summer demand)
- Top suppliers are major beer importers (Anheuser Busch, Crown Imports, etc.)

---

## 📁 Dataset Information

- **File Name**: `Retail and warehouse Sale.csv`
- **Rows**: 30,000+
- **Columns**: 9
- **Time Period**: 2020 (Months: January, March, July, September)
- **Main Categories**: Beer, Wine, Liquor, Kegs, Supplies

### Columns
| Column              | Description                        |
|---------------------|------------------------------------|
| YEAR                | Year of sale                       |
| MONTH               | Month of sale                      |
| SUPPLIER            | Supplier name                      |
| ITEM CODE           | Unique item identifier             |
| ITEM DESCRIPTION    | Product name                       |
| ITEM TYPE           | Category (BEER, WINE, LIQUOR, etc.)|
| RETAIL SALES        | Sales through retail channel       |
| RETAIL TRANSFERS    | Internal stock transfers           |
| WAREHOUSE SALES     | Bulk warehouse/distribution sales  |

---

## 📈 Key Metrics

- **Grand Total Sales**: $1,031,118
- **Retail Sales**: $208,186 (20.19%)
- **Warehouse Sales**: $822,931 (79.81%)
- **Top Category**: Beer (73.4%)
